PureBit User Guide

Quick Start

  1. Install the BM Mic driver package first.
  2. Launch PureBit.
  3. Select a source microphone.
  4. Verify that the output target shows the BM Mic virtual endpoint.
  5. Press START BM MIC STREAM.

Controls

  • Noise Reduction Strength controls the RNNoise blend.
  • Output Volume Gain applies post-processing gain.
  • Echo settings expose the built-in reverb stage.
  • Plugins Manager lets you enable, disable, and configure DSP plugins.
  • Recording captures the processed result to a 48 kHz WAV file.

Operational Notes

  • The BM Mic driver must remain installed and visible to Windows.
  • Some plugins require large local assets and may only be available in the full runtime package.
  • If the stream fails to start, verify that the BM Mic endpoint exists and that Windows audio services are healthy.
