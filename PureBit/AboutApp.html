PureBit

PureBit is the Windows-side real-time processing engine used with the BM Mic driver stack. It captures a source microphone, applies live DSP and plugin effects, then injects the processed stream directly into the BM Mic virtual driver.

The runtime is optimized for low-latency monitoring, plugin-based effects, and a direct virtual-microphone output path designed for applications that expect a standard Windows capture endpoint.

If the app sounds suspiciously good, blame the plugins. If it sounds suspiciously strange, also blame the plugins.
