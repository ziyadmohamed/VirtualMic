PureBit Addons Guide

PureBit discovers plugins from the local plugins directory. Each plugin is a Python module that exposes a Plugin class with a process() method.

Basic Rules

  • Keep plugins frame-count stable. Return the same number of samples you receive.
  • Use mono processing unless the plugin explicitly supports stereo.
  • Persist plugin state through the shared settings file when needed.

Deployment

  • Portable builds can accept new plugin files directly under the local plugins folder.
  • Install builds preserve user plugin files during upgrades whenever possible.
  • Large asset-based plugins may be excluded from compact runtime packages to keep installer size under control.
